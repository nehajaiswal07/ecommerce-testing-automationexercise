# E-commerce Testing Project (AutomationExercise)

This project demonstrates manual testing of an e-commerce web application using real-world scenarios.

## 🔍 Project Overview
The application tested is AutomationExercise, covering major modules like Login, Signup, Product, Cart, and Checkout.

## 🧪 Testing Scope
- Functional Testing
- Regression Testing
- UI Testing
- End-to-End Testing

## 📋 Test Scenarios Covered
- User Registration and Login
- Product Search and View
- Add to Cart / Remove from Cart
- Checkout Process
- Form validations and error handling

## 📊 Test Cases
- Designed and executed 50+ test cases
- Covered positive and negative scenarios
- Included edge case validations

## 🐞 Bug Reporting
- Identified and documented defects
- Included severity and priority levels
- Provided steps to reproduce and expected vs actual results

## 🛠 Tools Used
- MS Excel (Test Cases)
- JIRA (Bug Tracking - simulated)
- Browser Testing

## 📁 Project Files
- TestCases.xlsx
- BugReport.docx

## 🎯 Key Learning
- Practical understanding of SDLC & STLC
- Hands-on experience in real-time application testing
- Improved bug identification and reporting skills
